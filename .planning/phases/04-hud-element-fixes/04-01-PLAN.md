**MenuGridUI.luau -- Mobile scaling (per MENU-01 and MENU-02):**

1. Add `local UserInputService = game:GetService("UserInputService")` at top with other services.

2. Add a mobile detection constant after the existing layout constants block:
```luau
local IS_MOBILE = UserInputService.TouchEnabled
```

3. Add mobile-specific layout constants below the existing constants. On mobile, use 0.65x scale factor applied to button sizes and panel dimensions. This keeps buttons at ~48x44 which satisfies the 44px+ tap target requirement (MENU-02):
```luau
local MOBILE_SCALE = 0.65
local M_BUTTON_W = if IS_MOBILE then math.floor(BUTTON_W * MOBILE_SCALE) else BUTTON_W  -- 48 on mobile
local M_BUTTON_H = if IS_MOBILE then math.floor(BUTTON_H * MOBILE_SCALE) else BUTTON_H  -- 44 on mobile
local M_GAP = if IS_MOBILE then 4 else GAP
local M_PAD = if IS_MOBILE then 6 else PAD
local M_PANEL_W = M_PAD * 2 + COLS * M_BUTTON_W + M_GAP  -- ~114 on mobile vs 172 on desktop
local M_PANEL_H = M_PAD * 2 + 4 * M_BUTTON_H + M_GAP * 3 -- ~200 on mobile vs 306 on desktop
local M_PANEL_Y_OFFSET = -math.floor(M_PANEL_H / 2)
```

4. In `MenuGridUI.new()`, replace all references to `PANEL_W`, `PANEL_H`, `PANEL_Y_OFFSET` with the M_ prefixed versions in the panel creation call:
```luau
size = UDim2.new(0, M_PANEL_W, 0, M_PANEL_H),
position = UDim2.new(0, PANEL_X, PANEL_Y_SCALE, M_PANEL_Y_OFFSET),
```

5. In the grid button loop, replace `BUTTON_W`, `BUTTON_H`, `GAP`, `PAD` with `M_BUTTON_W`, `M_BUTTON_H`, `M_GAP`, `M_PAD`:
```luau
local x = M_PAD + col * (M_BUTTON_W + M_GAP)
local y = M_PAD + row * (M_BUTTON_H + M_GAP)
```
And update button size:
```luau
size = UDim2.new(0, M_BUTTON_W, 0, M_BUTTON_H),
```

6. Scale down icon and label text sizes on mobile. In the Icon label creation:
```luau
textSize = if IS_MOBILE then 18 else 24,
```
In the Name label creation:
```luau
textSize = if IS_MOBILE then 10 else 12,
```

7. In `hide()` and `show()` methods, update the position targets to use `M_PANEL_W` and `M_PANEL_Y_OFFSET`:
- hide: `UDim2.new(0, -M_PANEL_W - 10, PANEL_Y_SCALE, M_PANEL_Y_OFFSET)`
- show: `UDim2.new(0, PANEL_X, PANEL_Y_SCALE, M_PANEL_Y_OFFSET)`
