    With scale-aware version (per D-04 — manual CanvasSize, dividing AbsoluteContentSize by UIScale.Scale):
    ```luau

        local scaleFactor = uiScaleObj and uiScaleObj.Scale or 1
        self.listFrame.CanvasSize = UDim2.new(0, 0, 0, layout.AbsoluteContentSize.Y / scaleFactor + 12)
    end)
    ```

    This ensures CanvasSize uses unscaled content size even when UIScale is active.
